## 📦 Nima uchun bu repo?

**nadpos.apps** — NadPOS Restoran (POS tizimi) uchun rasmiy **relizlar** ombori. Dastur ichidagi **Sozlamalar → Yangilash** bo‘limi yangilanishlarni aynan shu repodan tekshiradi va yuklab oladi.

| Imkoniyat | Tavsif |
|-----------|--------|
| 🔄 **Avtomatik yangilanish** | Dastur ichidan «Yangilanishni tekshirish» orqali yangi versiyani olish |
| 📥 **Qo‘lda o‘rnatish** | Releases bo‘limidan `.exe` ni yuklab, qo‘lda o‘rnatish mumkin |
| 📋 **Versiya tarixi** | Barcha relizlar va o‘zgarishlar ro‘yxati |

---

## 🚀 Yangilanishni qanday olish mumkin?

### Variant 1: Dastur ichidan (tavsiya etiladi)

1. NadPOS Restoran dasturini oching  
2. **Sozlamalar** (⚙️) → **Yangilash** bo‘limiga o‘ting  
3. **«Yangilanishni tekshirish»** tugmasini bosing  
4. Agar yangi versiya bo‘lsa — **Yuklash** va keyin **Qayta ishga tushirish** orqali yangilang  

### Variant 2: GitHub orqali qo‘lda

1. [Releases](https://github.com/just-nadir/nadpos.apps/releases) sahifasiga o‘ting  
2. Eng so‘nggi relizdan **NadPOS-Restoran-Setup-*.exe** faylini yuklab oling  
3. O‘rnatuvchini ishga tushiring va ko‘rsatmalarga amal qiling  

---

## 📌 Relizlar tuzilmasi

Har bir relizda odatda quyidagilar bo‘ladi:

- **NadPOS-Restoran-Setup-{versiya}.exe** — Windows uchun o‘rnatuvchi (NSIS)
- **RELEASE_NOTES** — o‘zgarishlar va yangilanishlar haqida qisqacha ma’lumot  

Versiya formati: `X.Y.Z-N` (masalan, `3.1.6-1`).
